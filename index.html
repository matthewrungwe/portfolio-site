Mobile Web Specialist

This is a Google coined program. Focuses on the best practices to designing and developing web applications that are responsive and accessible. It goes further into ways of optimising and improving the overall perfomance of our web applications. Other details...
  • Responsive Web design: concepts that improve the performmance of webpages on small devices. Allows users on small devices to have the same experience as users on bigger devices.
  • Mobile Networks: detailed dive into using fetch() to attain resources. Multiple ways of validating reponses, handling successful responses and catching errors.
  • Accessibility: Deeper overview of how to use ARIA to make webpage elements meaningful to assisstive technology. Semantics is discussed as a close alternative to ARIA and developing meaningful and flowing webpages.
  • Progressive Web Apps: Introduction to service workers, active scripts on the website that improve the behaviour and minimise page load and isolating the application shell.
  • Exposure to HTML, CSS, JavaScript and Angular IQ tests. Recommended practice and advanced concepts.
